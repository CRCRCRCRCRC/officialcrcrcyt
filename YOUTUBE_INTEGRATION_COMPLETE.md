## 🚀 使用方式

### 1. 設置 YouTube API（可選）
如果要使用真實的 YouTube 數據，請按照 `YOUTUBE_SETUP.md` 指南設置 API 金鑰。

### 2. 環境變數配置
在 `backend/.env` 文件中：
```env
YOUTUBE_API_KEY=你的API金鑰（可選）
YOUTUBE_CHANNEL_ID=頻道ID（可選）
YOUTUBE_CHANNEL_HANDLE=officialcrcrcyt
```

### 3. 測試配置
運行測試腳本：
```bash
cd backend
node test-youtube.js
```
