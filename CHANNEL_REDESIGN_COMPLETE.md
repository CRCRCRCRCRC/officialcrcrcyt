## 🚀 YouTube API 設置指南

### 1. 獲取 YouTube API 密鑰

1. 前往 [Google Cloud Console](https://console.cloud.google.com/)
2. 創建新項目或選擇現有項目
3. 啟用 YouTube Data API v3：
   - 在左側菜單中選擇「API 和服務」→「庫」
   - 搜索「YouTube Data API v3」
   - 點擊並啟用

4. 創建 API 密鑰：
   - 在左側菜單中選擇「API 和服務」→「憑據」
   - 點擊「創建憑據」→「API 密鑰」
   - 複製生成的 API 密鑰

### 2. 獲取頻道 ID

**方法一：從頻道 URL 獲取**
- 如果你的頻道 URL 是 `https://youtube.com/@officialcrcrcyt`
- 使用工具：https://commentpicker.com/youtube-channel-id.php

**方法二：從 YouTube Studio**
1. 登入 YouTube Studio
2. 在左側選單點擊「設定」
3. 選擇「頻道」→「進階設定」
4. 複製「頻道 ID」

**方法三：從頻道頁面源代碼獲取**
- 打開你的頻道頁面
- 查看頁面源代碼，搜索 `"channelId"`

### 3. 配置環境變數

編輯 `backend/.env` 文件：

```env
# YouTube API 設定
YOUTUBE_API_KEY=你的_YouTube_API_密鑰
YOUTUBE_CHANNEL_ID=你的_頻道_ID
```

### 4. 測試 API

重啟後端服務器後，訪問：
```
http://localhost:3001/api/channel/youtube-data
```
