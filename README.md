# CRCRC 官方網站

CRCRC YouTube 頻道的官方網站，展示空耳音樂作品和頻道資訊。

## 功能特色

- 🎵 影片展示和管理
- 🎨 現代化響應式設計
- 🔐 後台管理系統
- 📱 手機端適配
- 🚀 快速載入

## 技術棧

- **前端**: React 18 + Vite + Tailwind CSS
- **後端**: Node.js + Express
- **資料庫**: SQLite
- **認證**: JWT Token

## 快速開始

### 安裝依賴
```bash
npm run install:all
```

### 開發模式
```bash
npm run dev
```

### 生產建置
```bash
npm run build
```

## 項目結構

```
├── frontend/          # React 前端應用
├── backend/           # Node.js 後端 API
├── database/          # SQLite 資料庫文件
└── docs/             # 文檔
```
